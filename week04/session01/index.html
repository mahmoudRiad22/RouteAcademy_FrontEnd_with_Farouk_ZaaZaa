6- media-query

description:
simply apply the media-classes into the item-parent. for now we only use width until we see whats gonna appear during the next project.


-- Example of a single screen media-query --

/* mobile-small ms- */
@media screen and (min-width: 320px) {
.ms-1 {
width: calc(100% / 1);
}

.ms-2 {
width: calc(100% / 2);
}

.ms-3 {
width: calc(100% / 3);
}

.ms-4 {
width: calc(100% / 4);
}

width: 20%;
width: calc(100% / 5);
}

}
}
