1-Hoisting

    What Does it do?

    • Hoisting for Variables:
    1. Takes the Decleration of variables to the top of the code scope.
    2. Keeps the Assignment in its place.

    • Hoisting for Functions:
      1. Hoisting For Decleration Functions:
      1. Takes teh full decleration to the top of the code scope.
      2. keeps the call of the function in place.

      2. Hoisting for Expression Functions:
      1. Takes the decleration of the variable to top of the code scope.
      2. Keesps the Assignment and calls in its place.


2-Objects and Arrays

Objects:


  • Syntax:
    let object1 = {};
  • Example:
let userData = {
name: "mahmoud",
age: 31,
email: mahmoud@gmail.com
};
  • Acess:
    console.log(userData.name);
    console.log(userData['name']);

Arrays


    Syntax:
  • let array1 = [];
  • Examples:
// array of strings
let array1 = ['mahmoud', 'islam', 'mohamed', 'aya', 'nada'];

//array of numbers
let arrayOfNumbers = [30, 5, 18, 64, 69, 100];

//array of Objects

Arrays Acess


    Acess Syntax
  • Example:
let array1 = ['mahmoud', 'islam', 'mohamed', 'aya', 'nada'];

console.log(array1[3]); ====>// output: aya
console.log(array1[0]); ====>// output: mahmoud
console.log(array1[4]); ====>// output: nada

    Looping an Array:


  1. Using an array method "foreach".
    Example
let array1 = ["mahmoud", "islam", "mohamed", "aya", "nada"];

array1.forEach((item) => {
    console.log(item);
});
Output:
mahmoud
islam
mohamed
aya
nada

  2. Using a "for...of".
    Example:
let array1 = ["mahmoud", "islam", "mohamed", "aya", "nada"];

for (const item of array1) {
    console.log(item);
}
Output:
mahmoud
islam
mohamed
aya
nada

4-Array Methods

  1. .sort((s,b)=>(s-b)).... used to sort elements from small to big or reverse.
    [return sorted array]

  2. .reverse().... used to reverse the order of the array.
    [return reversed array]

  3. .push(item).... used to add new element at the end.
    [return new length]

  4. .unshift(item).... used to add new element at the start.
    [return new length]

  5. .pop().... removes last element of array.
    [return the removed element]

  6. .shift().... removes the 1st element of array.
    [returns removed element]

  7. .at(index).... return the value at given index.

  8. .slice(startindex, end index excluded).... return sub-array based on start and end indexes.

  9. .splice(start index included, deleteCount, ...items).... full manipulation of an array (add, remove, replace).
    [return updated array]

  10. .indexof('value').... return the index of 1st match if found or -1 if not found.

  11. .lastIndexOf('value').... return the index of last match if found oor -1 if not found.

  12. .includes('string').... return True if atleast one match if found otherwise False

  13. .toString().... return a string with all array elements sperated by ','.

  14. .join('seperator').... return a string with all array elements seperated by given seperator.
