1- For Loop

types of loops:



  1. Normal Loops:
    1. For ( ; ; ) {}
    2. while() {}
    3. do {} while ()


  2. Special Use Loops:
    1. for (const value of iterable){}; =====> Use for: Arrays, strings, maps, sets (iterable objects).
    2. for (const key in object) {}; ======> Use for: Object properties (not recommended for arrays).
    3. array.foreach( (value, index, array) =>{}); : Use for: Arrays only (built-in method).
