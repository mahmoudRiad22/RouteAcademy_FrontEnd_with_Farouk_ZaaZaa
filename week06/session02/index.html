1-containers

description: there are build in classes in bootstrap.

.container: apply a fixed width in each breakpoint.
.container-sm: apply the container from the sm screen and above.
.container-md: apply the container from the md screen and above.
.container-lg: apply the container from the large screen and above.
.container-xl: apply the container from the extra large screena and above.
.container-xxl: apply the container from the xx-large screen and above.
.container-fluid: no container is used just a simple padding of 12px left and right.
