2- line-height, letter-spacing, text-transform, scroll-behavior.

description:
inline-height: allow you to adjust the height in between lines, will be given using font extensions.
letter-spacing: allow you to adjust the sapce inbetween letters, can use a [1.2] value which is multiplied by the font-size.
text-trasform: allow you to change the text style into [Capitalized, lowercase, uppercase, none]
scroll-behavior: allow you to make the scroll between sections to be smooth.. and always given to html element.
