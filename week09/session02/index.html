1-Mathmatical Operators

Arithmatic Opertators:
  • plus ==> [ + ]
  • minus ==> [ - ]
  • multiply ==> [ * ]
  • divide ==> [ / ]
  • inc ==> [ ++ ]
  • dec ==> [ -- ]
  • power ==> [ ** ]
  • moduls ==> [ % ]
Assign Operators:
  • assign ==> [=]
  • add then assign ==> [+=]
  • minus then assign ==> [-=]
  • multiply then assign ==> [*=]
  • divide the assign ==> [/=]
Comparison Operators:
  • is equal to ==> [==]
  • is greater than ==> [>]
  • is greater than or equal to ==> [>=]
  • is less than ==> [<]
  • is less than or equal ==> [<=]
  • is identical to ==> [===]
  • is NOT equal to ==> [!=]
  • is NOT identical ==> [!===]
Logical Operators:
  • AND ==> [ && ]
  • OR ==> [ || ]
  • NOT ==> [ ! ]
Priority:
  1. ()
  2. !
  3. &&
  4. ||
Logical Operators Examples:
  • true && true ====> true
  • true && false ====> true
  • false && false ====> false

  • true || true ====> true
  • true || false ====> true
  • false || false ====> false

  • !true ====> flase
  • !false ====> true
Falsy Values:
    always return "FALSE"
  • 'false'
  • ' 0 '
  • ' -0 '
  • ' undefined '
  • ' null '
  • ' NaN '
  • ' "" empty string'
Explicit Conversion from String to Number:
  • unary sign ===> " + "
  • Number()
Applying-section:

Implicit Conversion from String to Number:
  • "100" ===> 10
  • "" ===> 0
  • " " ===> 0
  • null ===> 0
  • false ===> 0
  • true ===> 1
  • "string value" ===> NaN
  • undefined ===> NaN

  • "if" and "if else" statments
