how to hide elements

description: 

           1- Display:none;        // totally hide the element from rendering at all
           2- visiability: hidden; // hide the element content after rendering it
           3- opacity: 0~1;        // the element is rendered but makes the element tansparent based on its value
                                   // you can still see the hover effect
           4- hidden               // a global html attribute used to make a the element hidden
